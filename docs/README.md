## Contributing to Documentation

When updating documentation:
- Keep the main [README.md](../README.md) in the root directory
- Place all detailed documentation in this `docs/` directory
- Update cross-references when moving or renaming files
- Use relative links for internal documentation references

## Need Help?

If you can't find what you're looking for:

2. Review the [QUICKSTART.md](QUICKSTART.md) guide
3. Consult [BUILD_NOTES.md](BUILD_NOTES.md) for common issues
4. Open an issue in the repository
